== Description ==

Provides ACF (Advanced Custom Fields) admin interface for creating custom post types.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Visit "Post Types" tab in the WP Admin to add a new custom post type.

== Screenshots ==

1. Post Types is added to the left admin menu
2. All registered Custom Post Types are listed like other posts in the admin
3. Editing an existing post type with the tabbed interface
4. Menu settings for post type, here you can set the position in menu and dashicon
5. ACF Post Types supports every type of label, and this is organized into it's own settings tab so it does not clutter the other setting areas
6. ACF Post Types is one of few post type plugins that supports custom rewrites
7. Advanced settings includes options including the "supports" settings
